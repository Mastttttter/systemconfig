## 我使用的是YASB+gravalWM

首先在

https://github.com/amnweb/yasb

https://github.com/glzr-io/glazewm

选择一个方式下载

glazewm需要按照文档的指示安装<insert layout>https://github.com/Dutch-Raptor/GAT-GWM

然后将

``` YASB ```复制到``` windows-user/.config ``` 下

``` .glzr ```复制到 ```windows-user/ ```下

将glazeWM 的快捷方式放在启动文件夹下 (如何打开： 运行``` shell:startup ```` )

## windowsTerminal

打开任意一个windowsTerminal 按``` ctrl+, ```进入设置，点击左下角 “打开JSON文件”

将./windowsterminal/settings.json复制进去
